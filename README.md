# Personal Dashboard

**Personal Dashboard** é uma aplicação web interativa para gerenciamento pessoal de tarefas, notas e metas. Esta é a **primeira versão** e ainda pode conter alguns erros ou comportamentos inesperados.  

---

## Funcionalidades

- **Notas**: Adicione, visualize e edite suas notas de forma simples.  
- **Tarefas**: Organize tarefas em um calendário **drag & drop** para facilitar o planejamento.  
- **Metas**: Defina metas com cores que representam a importância ou o peso de cada ação.  

---

## Tecnologias

- HTML / CSS / JavaScript  
- Python (para servidor local)  

---

## Como rodar localmente

1. Clone o repositório:
   git clone https://github.com/DaviReder/Personal-Dashboard.git
   
2. Acesse a pasta do projeto:
   cd Personal-Dashboard

3. Inicie o servidor local:
  python -m http.server 8000

4. Abra o navegador e acesse:
  http://localhost:8000

⚠️ Lembre-se: esta é a primeira versão, então podem haver bugs ou melhorias necessárias.
